Featured Project
Help Desk AI Toolkit
Two LLM-powered tools for an IT help desk on a shared classification core: a Tier-1 triage agent that reads a ticket, pulls knowledge-base context, and drafts a response for human approval, and a bulk categorizer that cleans up a backlog with a two-stage keyword-then-model pipeline. The headline design choice is a hybrid model architecture — a frontier model for hard reasoning, a local open-source model for cheap high-volume work — exposed as a single config switch.
